	def showScoreDB(scdb, keyname):
    	for p in sorted(scdb, key=lambda person: person[keyname]):
        	for attr in sorted(p):
            	print(attr + "=" + p[attr], end=' ')
        	print()
=>사람들의 이름을 알파벳기준으로 재정렬 하여 정보를 보여주는 함수입니다.
